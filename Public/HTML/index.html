Tutorial

Como jogar:


Jquery Game é um jogo no estilo, do inglês, "typer" onde, o jogador(a), deve redigir o texto exibido no local indicado. O objetivo é escrever o maior número de palavras possíveis dentro do tempo fornecido. Ou, se você for rápido o suficiente, todo o texto (fica o desafio 😀).

Escolhendo a dificuldade:


No Jquery Game, é possível escolher a dificuldade do que você, o jogador(a), irá "enfrentar"! A dificuldade influencia a complexidade, tamanho do texto, tempo fornecido e a adição de desafios.

Desafios:


Desafios são palavras com propriedades especiais. Elas são indicadas por cores e cada uma tem a sua peculiaridade, como:

Desafio 1: A primeira letra da palavra maiúscula.
Desafio 2: A palavra deve ser escrita duas vezes em sequencia.
Desafio 3: Toda a palavra deve ser maiúscula.
Desafio 4: A palavra deve ser escrita invertida.
Desafio 5: A ultima letra da palavra minúscula.

 Entendi!
